# JSON Web Token with Redis

This is Login feature for practice JWT and Redis.

<br>

## **Install node_modules**

- express
- jsonwebtoken
- nodemon
- redis

`npm i express jsonwebtoken nodemon redis`

<br>

## **Redis control**

```shell
$> redis-cli

127.0.0.1:6379> set counter 0

127.0.0.1:6379> get counter
"0"

```
